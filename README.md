# Rossby

## Usage
``rossby.py [-h] --input INPUT [--output OUTPUT] [--n N] [--log LOG] [--e-mail E_MAIL]``

``  -h, --help       show help``

``  --input INPUT    Input file name``

``  --output OUTPUT  Output file name``

``  --n N            Number of harmonics``

``  --log LOG        Log file``

``  --e-mail E_MAIL  E-mail``
